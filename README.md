## Задание 1

### Текст задачи

Указать все делители натурального числа n.

### Алгоритм для задания 1

1. Получить натуральное число n от пользователя
2. Для каждого числа m от 1 до n:
   - Проверить, делится ли n на m без остатка (n mod m = 0)
   - Если делится, добавить m в список делителей
3. Вывести полученный список

## Задание 2

### Текст задачи

Найти сумму чётных элементов списка целых чисел.

### Алгоритм для задания 2

1. Получить список целых чисел от пользователя
2. Если список пуст, сумма равна 0
3. Иначе:
   - Найти сумму чётных элементов хвоста списка
   - Если первый элемент чётный, прибавить его к сумме хвоста
   - Если нечётный, сумма равна сумме хвоста
4. Вернуть итоговую сумму

## Задание 3

### Текст задачи

Доказать закон де Моргана для объединения множеств: complement(A ∪ B) = complement(A) ∩ complement(B).

### Алгоритм для задания 3

1. Получить универсальное множество U, множества A и B от пользователя
2. Вычислить объединение A и B: A ∪ B
3. Вычислить дополнение объединения: U \ (A ∪ B)
4. Вычислить дополнение A: U \ A
5. Вычислить дополнение B: U \ B
6. Вычислить пересечение дополнений: (U \ A) ∩ (U \ B)
7. Сравнить полученные множества
8. Вывести результат проверки закона

## Задание 4

### Текст задачи

Для полярной экспедиции из восьми претендентов А, В, С, D, Е, F, G и Н надо отобрать шестерых специалистов: биолога, гидролога, синоптика, радиста, механика и врача. Обязанности биолога могут выполнять Е и G, гидролога — В и F, синоптика — F и G, радиста — С и D, механика — С и Н, врача — А и D. Хотя некоторые претенденты владеют двумя специальностями, в экспедиции каждый сможет выполнять только одну обязанность. Кого и кем следует взять в экспедицию, если F не может ехать без В, D — без С и без Н, С не может ехать одновременно с G, а A вместе с B?

### Алгоритм для задания 4

1. Сгенерировать все возможные комбинации из 6 кандидатов из 8
2. Для каждой комбинации:
   - Проверить возможность назначения уникальных специальностей каждому кандидату
   - Проверить выполнение ограничений:
     - F не может ехать без B (если F в команде, то B тоже должен быть)
     - D не может ехать без C и без H (если D в команде, то C и H тоже должны быть)
     - C не может ехать одновременно с G (C и G не могут быть вместе)
     - A не может ехать вместе с B (A и B не могут быть вместе)
3. Вывести все найденные решения с распределением специальностей
